# Cloud Products Supporting CLI

Updated:

This article lists the cloud products currently supported by Alibaba Cloud CLI and their corresponding Codes. You can also get the latest supported product information through the following methods:

- Execute `aliyun --help` command to dynamically query the list of supported products
- View the latest supported product information through [GitHub Repository](https://raw.githubusercontent.com/aliyun/aliyun-openapi-meta/master/metadatas/products.json)

**Note**

This document content may have some lag (last updated: 2025-10-31), it is recommended to prioritize getting real-time information through the above dynamic methods.
